ACTIVIDAD TRES

Realice un algoritmo que, con base en una calificación proporcionada (0-10), indique con letra la calificación que le corresponde: 10 es “A”, 9 es “B”, 8 es “C”, 7 y 6 son “D”, y de 5 a 0 son “F”. Represente el diagrama de flujo y Código JS correspondiente.

Resolucion simple

let calificacion = parseInt(prompt("Digite la calificacion obtenida")); switch (calificacion) { case 10: nota = "A"; break; case 9: nota = "B"; break; case 8: nota = "C"; break; case (7, 6): nota = "D"; break; default: nota = "F"; } alert(calificacion)
